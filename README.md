# Java Projects 

A collection of Java console applications built using Core Java and OOP concepts.

## Projects

### 1. ATM Simulator 
A console-based ATM simulation system.

**Features:**
- Check Balance
- Deposit Money
- Withdraw Money
- Insufficient balance detection
- Exit option

**Concepts used:** OOP, Encapsulation, Methods, Loops, Switch

---

### 2. Vending Machine 
A console-based Vending Machine simulation.

**Features:**
- 8 items with different prices
- Change calculation
- Insufficient amount detection
- Continuous shopping with exit option

**Concepts used:** Methods, Switch, Loops, Scanner

---

## How to Run

1. Clone the repository
2. Open in any Java IDE (VS Code / IntelliJ)
3. Run `ATM.java` or `W_machine.java`
